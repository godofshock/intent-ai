# AI Developer Intent Inference Engine

A deep learning system that predicts developer intent from partial code input.

Unlike traditional autocomplete systems, this project focuses on understanding *what the developer is trying to do*, not just predicting the next token.

## Features

* Code-aware tokenization
* Sequence-based dataset generation
* LSTM-based intent classification model
* Real-time intent prediction

## Example Intents

* Loop construction
* Conditional logic
* Function definition

## Tech Stack

* Python
* PyTorch
* NLP for code

## Status

✅ Completed (v1)

## Goal

To explore how AI can move beyond syntax prediction and begin understanding developer intent in real-time coding scenarios.

## Future Improvements

* Transformer-based architecture
* Larger dataset from real codebases
* IDE integration
